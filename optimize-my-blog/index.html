主要记录一下对自己的博客的改造过程.

内联部分重点 css

总体步调

  1. 速度:
    • 请求数: 页面初始请求不超过6个
    • 请求大小: 最大请求大小不超过150k
    • 不超过2s 加载完毕
    • 静态文件采用 cdn 缓存
    • 图片懒加载
    • 干掉 jquery
  2. 样式
    • 整体风格天蓝色
    • 采用卡片式布局(可以考虑重写, 既可以练习自己的建站技巧, 也可以去除原本 css 中的冗余部分)
  3. 安全
    • 全站升级为 https
    • 七牛云空间也一并配置 https
  4. 评论
    • 等到前三点稳定以后再开启评论合适/速度快/国内可访问的评论系统
  5. 搜索
    • 等到前四点稳定以后再考虑转用其他搜索实现毫秒级响应(第三方搜索或者自己部署 Elasticsearch)
      目前采用的是hexo-generator-search生成的站内静态文件.
